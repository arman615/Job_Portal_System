# Job Portal System
The Job Portal System is a C++ program that simulates a simple job portal where users can register jobs, apply for jobs, and receive job notifications based on their qualifications.

# Features
.Job Registration: Employers can register job openings with details like job title, description, requirements, salary, and company name.


.User Registration: Job seekers can register with their name, contact information, and resume.


.Job Search: Users can search for open jobs based on keywords in job titles, descriptions, and requirements.


.Job Application: Users can apply for open jobs.


.Job Closure: Employers can close job openings.


.Job Notifications: Users receive job notifications for open positions that match their resume.
